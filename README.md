Análisis de datos detallado para la identificación de métricas clave y patrones de comportamiento. Incluye procesos de limpieza, ETL y visualización de resultados con Python.
# Análisis de Datos - Proyecto ConnectaTel

## Objetivo del Proyecto
Analizar el comportamiento de los usuarios de ConnectaTel para identificar patrones de deserción (churn) y proponer estrategias basadas en datos que mejoren la retención de clientes.

## Datasets Utilizados
* `telecom_users.csv`: Contiene información demográfica, servicios contratados y registros de facturación de los clientes.

## Etapas del Análisis
1. **Preprocesamiento:** Limpieza de datos, manejo de valores ausentes y conversión de tipos de datos.
2. **Análisis Exploratorio (EDA):** Visualización de distribuciones y correlaciones entre variables clave.
3. **Modelado y Métricas:** Evaluación de indicadores de desempeño (KPIs) para entender el abandono de clientes.

## Cómo ejecutar el notebook
Para visualizar y ejecutar el análisis:
1. Descarga el archivo `.ipynb` de este repositorio.
2. Súbelo a [Google Colab](https://colab.research.google.com/) o ábrelo localmente con Jupyter Notebook.
3. Asegúrate de cargar el dataset correspondiente para que las celdas se ejecuten correctamente.

## Guía de reproducción
1. Instalar las librerías necesarias: `pandas`, `matplotlib`, `seaborn` y `plotly`.
2. Ejecutar las celdas en orden secuencial para procesar los datos y generar las visualizaciones.
